### Requirements

* node  
* react-native-cli   
* xcode  
* react-web  

### Ios Step 1
```
$ git clone https://github.com/febobo/react-native-redux-es6.git
$ cd react-native-redux-es6
$ npm install
$ npm start
```

### Ios Step 2
使用xcode 打开react-native-redux-es6/ios/demo.xcodeproj  
or ```$ open demo.xcodeproj```

### Web Start
```
$ react-web start
open http://127.0.0.1:3000
```

### PS
只完成基本算法，仅供学习参考使用
